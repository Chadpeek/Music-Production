# Support for Producer OS

Thank you for using Producer OS!  If you run into any issues or
have suggestions for new features, we’re here to help.

## Getting Help

1. **Search existing issues** – Before opening a new issue, please
   search the [issue tracker](https://github.com/KidChadd/Producer-OS/issues)
   to see if your question has already been answered.

2. **Open a bug report** – If you discover a problem, please open a
   bug report.  Include a clear description of the problem, steps
   needed to reproduce it, and any relevant logs (found under
   ``hub/logs/<run_id>``).

3. **Request a feature** – Have an idea for improving Producer OS?
   Open a feature request describing the problem you’re trying to
   solve and how Producer OS could help.

4. **Join the discussion** – We love hearing from our users.
   Participate in discussions on GitHub or open a new topic.

## Security

If you discover a security vulnerability in Producer OS, please
report it privately by sending an email to the maintainer rather
than opening a public issue.  We take security seriously and will
address reports promptly.
